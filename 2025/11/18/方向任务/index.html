解析目标报文（逐行说明）

  •  POST ：请求方式。
  •  /api/graphs/fulltext-search?hahaha=123 ：请求路径+查询参数（ hahaha=123 是URL参数）。
  •  HTTP/1.1 ：HTTP协议版本。
  • 目标服务器的域名/IP。
  • 客户端的Cookie信息。
  • 客户端的浏览器/系统信息（这里是Windows 10+Edge浏览器）。
  • 浏览器的UA信息； ?0 表示不是移动设备。
  • 客户端能接受的响应数据格式（HTML）。
  • 表示当前请求是从哪个页面跳转来的（来源页）。
  • 客户端支持的压缩编码方式。
  • 客户端偏好的语言（ q 是权重，zh-CN优先级最高）。
  • 表示请求完成后关闭TCP连接。
  • 请求体的数据格式。
  • 请求体的字节长度（这里是26字节）。
  • POST请求的请求体（表单数据： indexName 的值是 github ， keyword 的值是 1 ）
