二、遇到的问题及其解决方法

1.

  • 问题1：本地有 index.html ，但 GitHub Pages打开是404
  • 原因：（1） 手动上传文件时把 index.html 写成了 index_html （文件名错误）（2）index.htm里面内容为空
  • 解决：（1） 把仓库里的文件名改回 index.html （2） 寻找错误，重新建public文档

2.

  • 问题2：Hexo生成了首页，但主题（Next）没生效
  • 原因：安装包有问题
  • 解决：重新安装

3.

  • 问题3：环境变量配置失败，装完Node.js后，命令行无法识别“node”命令。
  • 解决：重新配置环境变量，将Node.js安装目录下的“bin”文件夹路径添加到系统环境变量“Path”，重启命令行后即可。
